# verdict.ai ✨
Aplikacja webowa dla profesjonalnych pełnomocników, pomagająca w efektywnym wyszukiwaniu podobnych orzeczeń i ich analizie w formie rozmowy z dokumentem. Wersja przystosowana do pracy z orzecznictwem w sprawach o alimenty. Narzędzie zostało wykonane podczas trzeciej edycji konkursu [hack4law](https://hack4law.pl/).

## Zbudowano przy pomocy 
* [Typescript](https://www.typescriptlang.org/)
* [Next.js](https://nextjs.org/)
* [Langchain](https://www.langchain.com/)
* [Pinecone](https://www.pinecone.io/)
* [Tailwind CSS](https://tailwindcss.com/)
* [DaisyUI](https://daisyui.com/)
* [SQLLite](https://www.sqlite.org/index.html)
* [Storyset](https://storyset.com/job)
